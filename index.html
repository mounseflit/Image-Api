Image Analysis API Documentation

This API allows you to analyze images using Google's Gemini AI model. You can upload images or use the camera to capture images for analysis.

The API provides endpoints for uploading images, analyzing them, and retrieving results.

You can also use the API to generate variations of images based on specific prompts.

Explore the API endpoints below to get started!

API Base URL

https://aicrafters-image-api.vercel.app

POST /analyze-image/gemini

Analyzes images using Google's Gemini AI model.

Request Body Parameters

Parameter Type Default Description
base64Image string Required Base64 encoded image data
prompt string "Describe the image" Prompt for image analysis
model string "gemini-2.0-flash" Gemini model to use
temperature number 0.5 Controls randomness in the output

GET /uploadgemini

Returns the upload interface for Gemini image analysis.
